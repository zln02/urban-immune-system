# 학습 로그 (Learning Log)

> 프로젝트 진행 중 새로 배운 기술·개념·패턴을 기록. 기업 인턴 면접 및 자기소개서 작성 시 직접 활용.

## 파일 규칙
- 파일명: `LL-{번호}_{주제}.md` (예: `LL-001_okabe_ito_cud_palette.md`)
- 한 항목당 5분 읽기 분량
- 섹션: 무엇을 배웠나 · 언제·왜 · 어디에 적용 · 외부 근거 · 심사 질문 대비
