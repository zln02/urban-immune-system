# Team Authored Posts

> 각 팀원이 **직접 쓴 기술 포스트·설계 문서**. 교수님의 "박진영 원맨쇼" 우려를 해소하는 가장 빠른 증거.

## 규칙
- 각 팀원이 본인 폴더에 최소 1편 이상 (발표 전까지)
- 길이: 500~1200 단어
- 형식: 문제 정의 → 설계 의사결정 → 구현 코드 일부 → 결과·배운 점
- PR 커밋 시 **git blame 에 본인 이름** 찍혀야 유효
- 주제는 아래 제안 중 선택 또는 자유
