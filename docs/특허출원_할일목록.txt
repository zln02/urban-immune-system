STEP 3. 학교 산학협력단 상담 예약
----------------------------------------------------------------------

목표: 비용 절감 + 전문가 도움 받기
      (혼자 내면 100~200만원, 산학협력단 통하면 50~80% 절감)

할 일

  3-1. 학교 산학협력단 홈페이지 접속

  3-2. 발명신고서 양식 다운로드

  3-3. 아래 내용 작성해서 제출
       - 발명 제목
         (예: 비의료 다중 신호 교차검증 기반 감염병 조기경보 시스템)
       - 발명자 이름 목록 (팀원 전원)
       - 발명 내용 요약 (A4 1~2장)
       - 이미 공개된 날짜 (공지예외 주장 위해)

  3-4. 담당자와 상담 일정 잡기
       - 특허 가능성 검토 요청
       - 출원 비용 분담 방식 확인
       - 변리사 연결 요청

       - 전체 아키텍처 다이어그램
       저장 위치: docs/특허_도면/

       - 기존 방법의 문제점
         (병원 보고 방식은 2주 뒤에야 알 수 있다)
       - 우리 시스템이 해결한 방법
         (3계층 교차검증으로 1~3주 선행 탐지)
       - 기존 방법과 다른 점 3가지 이상

       - FAR: 0.16
       - Lead Time: 평균 6.47주
       - 검증 방법: 17지역 walk-forward 백테스트
       저장 위치: docs/특허_성능수치.txt

  4-4. 참고 논문 3편 이상 수집 (선행기술 대비용)
       - 구글 스칼라에서 "wastewater surveillance influenza" 검색
       - "pharmacy data epidemic forecasting" 검색

STEP 5. 특허청 출원 제출
----------------------------------------------------------------------

목표: 특허로(patent.go.kr)에서 공식 출원

할 일

  5-1. 특허로 회원가입 (개인 공인인증서 필요)

  5-2. 제출 서류 최종 확인
       - 특허출원서
       - 명세서 (발명 설명 + 청구항)
       - 요약서 (A4 1장)
       - 도면 파일 (PNG 또는 PDF)

  5-3. 출원서에 공지예외 주장 체크박스 반드시 선택

  5-4. 출원료 납부
       - 금액: 약 4~6만원
       - 대학생 감면 적용 여부 확인

  5-5. 출원번호 수령 및 보관
       (이때부터 "특허 출원 중" 문구 사용 가능)

담당: 이경준 (산학협력단 담당자와 함께 진행)
기한: 2026-06-07 이전 (중간발표 후 30일 내)
