## 🌐 웹 UI 로 할 일 (박진영 권한 필요)

박진영이 Admin 이라 **박진영과 화상·같이 앉아서** 진행:

### Branch Protection 설정
```
https://github.com/zln02/urban-immune-system/settings/branches
→ Add rule (main)
  [v] Require pull request before merging
      Required approving reviews: 2
      [v] Dismiss stale approvals
      [v] Require review from Code Owners
  [v] Require status checks:
      backend-lint, backend-test, pipeline-lint, ml-lint, frontend-lint, legacy-test,
      Trivy FS Scan, CodeQL SAST (python), CodeQL SAST (javascript-typescript), Bandit
  [v] Require linear history
  [v] Include administrators

→ Add rule (develop)
  [v] Require PR + 1 review
  [v] Require status checks (동일)
```

### Dependabot Alerts 활성화
```
https://github.com/zln02/urban-immune-system/settings/security_analysis
→ Dependabot alerts: Enable
→ Secret scanning: Enable
→ Push protection: Enable
```
