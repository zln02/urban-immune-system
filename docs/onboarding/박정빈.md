## ⚡ 30분 안에 완료 (3단계)

### 1️⃣ GitHub 웹 UI — Branch Protection (10분)

**박진영과 같이 앉아서** 진행 (Admin 권한 필요):

🔗 [Settings > Branches > Add rule](https://github.com/zln02/urban-immune-system/settings/branches)

**규칙 1: `main`**
```
Branch name pattern: main

[v] Require a pull request before merging
    Required approving reviews: 2
    [v] Dismiss stale pull request approvals when new commits are pushed
    [v] Require review from Code Owners

[v] Require status checks to pass before merging
    Required checks (체크박스 클릭):
      - backend-lint
      - backend-test
      - pipeline-lint
      - ml-lint
      - frontend-lint
      - legacy-test
      - Trivy FS Scan
      - CodeQL SAST (python)
      - CodeQL SAST (javascript-typescript)
      - Bandit (Python SAST)

[v] Require linear history
[v] Include administrators

Save changes 클릭
```

**규칙 2: `develop`** (동일하게, Required reviews 는 **1** 로)

### 2️⃣ Dependabot 활성화 (5분)

🔗 [Settings > Code security and analysis](https://github.com/zln02/urban-immune-system/settings/security_analysis)

```
Dependabot alerts               → Enable
Dependabot security updates     → Enable
Dependabot version updates      → 이미 Enable (dependabot.yml 있음)
Secret scanning                 → Enable
Push protection                 → Enable
```

## 🆘 자주 막히는 지점

| 증상 | 해결 |
|---|---|
| Settings 메뉴 안 보임 | Admin 권한 필요 — 박진영에게 요청 (`Collaborators` 페이지에서 역할을 Maintain → Admin 임시 승격 or 박진영과 화상 통화하며 작업) |
| Branch Protection 404 | 규칙 저장 안 됨. 빨간 "Create" 버튼 클릭 확인 |
| `gh api ... 404` | 위 규칙 저장 안 됐다는 뜻 |
| `pytest-cov` 없음 | `pip install pytest-cov` (이미 pyproject.toml 에 있음) |
| 커버리지 50% 미만 | 정상. 아직 테스트 많이 없음 — 이후 목표 70% |
