## 🆘 자주 막히는 지점

| 증상 | 해결 |
|---|---|
| SSH 터널 "Connection refused" | 서버에 Streamlit 안 떠있음. `ssh` 로 서버 들어가서 확인: `curl http://localhost:8501/` |
| `localhost:8501` 안 열림 | 로컬 포트 이미 점유 중. 다른 포트로: `ssh -L 8502:localhost:8501 ...` → 브라우저 `localhost:8502` |
| 탭 클릭하니 빨간 에러 박스 | Streamlit 자동으로 보여줌. 에러 전문 복사해서 Claude 에게 |
| PDF 다운로드 버튼 404 | `reportlab` 미설치 — 박진영한테 "설치 좀" |
| 한국어 폰트 깨짐 | 서버에 폰트 설치 안 됨 — 박정빈한테 |
