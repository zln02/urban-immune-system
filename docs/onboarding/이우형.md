## 🆘 자주 막히는 지점
- **Kafka 연결 실패**: `KAFKA_BOOTSTRAP_SERVERS=localhost:9092` `.env` 확인
- **토픽 없음 에러**: Producer 한 번 실행 → 자동 생성됨
- **DB 에 row 없음**: Consumer 가 실제로 poll 하고 있는지 로그 확인
- **중복 INSERT**: offset commit 안 됨 → `enable_auto_commit=False` 후 수동
