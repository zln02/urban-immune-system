# 주간 기여도 스코어보드

> `scripts/update_scoreboard.py` 가 git log 로부터 자동 집계. 매주 월요일 오전 박진영이 `/weekly-report WNN` 실행 시 갱신.

## 지표

| 항목 | 설명 |
|---|---|
| **커밋** | `git log --author` 기준 해당 주차 커밋 수 |
| **PR 머지** | `gh pr list --state merged --author` 기준 |
| **PR 리뷰** | 남긴 리뷰 코멘트 수 |
| **테스트 추가** | 해당 주차 `tests/` 디렉토리 추가·수정 LoC |
| **이슈 처리** | 본인 열거나 닫은 이슈 수 |
