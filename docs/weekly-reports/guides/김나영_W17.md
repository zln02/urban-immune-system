## 작업 1: 검증 탭(`src/tabs/validation.py`) JSON 연결

### 배경

현재 검증 탭은 F1, Precision 같은 수치가 Python dict에 **하드코딩**돼 있음. 박진영이 실제 학습 후 `ml/outputs/validation.json` 을 생성하면, 김나영이 그 파일을 읽어 화면에 띄우는 구조로 변경.

### 절차

```bash
git switch -c feature/nyk-validation-json
```

**`src/tabs/validation.py` 수정** — 기존 하드코딩 부분 교체:

```python
# 파일 상단 import에 추가:
import json
from pathlib import Path

# 기존 results = {...} 하드코딩 부분을 아래로 교체:
VALIDATION_JSON = Path(__file__).parent.parent.parent / "ml" / "outputs" / "validation.json"

@st.cache_data(ttl=60)
def load_validation_results() -> dict:
    """ml/outputs/validation.json 로드. 없으면 시뮬레이션 fallback."""
    if VALIDATION_JSON.exists():
        return json.loads(VALIDATION_JSON.read_text(encoding="utf-8"))
    return {
        "f1": 0.71,
        "precision": 1.00,
        "recall": 0.55,
        "source": "simulation_fallback",
        "note": "실제 학습 결과 미연결 상태",
    }

def render():
    st.header("교차검증")
    results = load_validation_results()
    if results.get("source") == "simulation_fallback":
        st.warning("⚠️ 현재 시뮬레이션 값 표시 중. 실제 학습 결과 연결 대기.")
    col1, col2, col3 = st.columns(3)
    col1.metric("F1-Score", f"{results['f1']:.2f}")
    col2.metric("Precision", f"{results['precision']:.2f}")
    col3.metric("Recall", f"{results['recall']:.2f}")
```

## 작업 2: 리포트 탭 PDF 다운로드 버튼

### 목적

### 절차

```bash

git switch -c feature/nyk-report-pdf
```

**`src/tabs/report.py` 수정** — 리포트 렌더링 아래에 추가:

```python
# 파일 상단 import:
from io import BytesIO
from reportlab.lib.pagesizes import A4
from reportlab.pdfgen import canvas

# render() 함수 안, 리포트 내용 출력 직후 추가:
def _to_pdf(title: str, body: str) -> bytes:
    buf = BytesIO()
    c = canvas.Canvas(buf, pagesize=A4)
    c.setTitle(title)
    # 간단 1페이지 출력 (한글 폰트는 서버에 나중에 등록)
    c.setFont("Helvetica-Bold", 16)
    c.drawString(50, 780, title[:60])
    c.setFont("Helvetica", 10)
    y = 740
    for line in body.splitlines():
        c.drawString(50, y, line[:100])
        y -= 15
        if y < 50:
            c.showPage()
            y = 780
    c.save()
    return buf.getvalue()

report_title = "Urban Immune System 경보 리포트"
report_body = "\n".join([
    "발령 시각: 2026-04-20",
    "위험도: GREEN",
    "요약: 금주 3-Layer 신호 정상 범위.",
    # 실제로는 박진영이 생성한 경보 내용 주입 예정
])

st.download_button(
    label="📄 PDF 다운로드",
    data=_to_pdf(report_title, report_body),
    file_name=f"uis_alert_report_{datetime.now():%Y%m%d}.pdf",
    mime="application/pdf",
)
```
