## 1. 이번 주 한 일

- **테스트 계획서 초안 작성**: 우리 시스템이 제대로 동작하는지 검증할 때 어떤 항목을 어떤 순서로 확인할지 정리. 기능 테스트, 성능 테스트, 보안 테스트 3가지 큰 카테고리로 구성.

- **ISMS-P 체크리스트 점검**: 정부 납품 시 필요한 **정보보호 인증** 항목 중 우리 프로젝트가 현재 몇 %를 만족하는지 확인. 감사 로그·CORS·시크릿 관리 등 이경준·박진영과 협의해 진행 상황 업데이트.
- **자동 점검 시스템 연결 확인**: GitHub 코드 제출 시 자동으로 돌아가는 검사 6가지(backend 문법 검사, 테스트, 보안 스캔 등)가 실제로 작동하는지 로그 확인.
