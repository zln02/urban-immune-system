## 1. 이번 주 한 일

- **대시보드(Streamlit: Python 코드로 웹 화면을 빠르게 만드는 도구) 5개 탭 점검**: 위험도 지도·시계열·상관관계·교차검증·AI 리포트 탭 모두 렌더링 되지만, 내부 수치가 고정값(하드코딩)이라 실제 데이터 연동이 없음을 확인.
- **검증 탭(`src/tabs/validation.py`) 개선 준비**: 박진영이 생성할 `ml/outputs/validation.json` 결과 파일을 읽어 화면에 표시하는 구조로 변경 예정.
- **리포트 탭(`src/tabs/report.py`) PDF 다운로드 기능 설계**: `st.download_button`을 사용해 경보 리포트를 PDF 파일로 저장하는 기능 추가 계획.
- **Next.js(React 기반 프론트엔드 프레임워크) Phase 2 프로젝트 구조 확인**: `frontend/` 디렉토리에 기본 페이지 1개와 컴포넌트 3개만 있고, Mapbox(지도 렌더링 서비스) 토큰 미설정 상태.
