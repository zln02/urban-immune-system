## 1. SW 저작권 등록 — 2만원, 온라인 신청

### 왜 해야 하나
- 코드 자체가 저작물로 보호됨
- 분쟁 시 등록일이 법적 증거
- 특허 출원 여부와 무관하게 즉시 효력

### 신청 방법

**사이트**: https://www.copyright.or.kr (한국저작권위원회 CopyrightKorea)

1. 회원가입 후 **저작물 등록** → **컴퓨터프로그램** 선택
2. 입력 정보:
   ```
   저작물명: Urban Immune System (비의료 이종 신호 교차검증 기반 감염병 조기경보 시스템)
   저작자: 박진영
   창작연월일: 2026-03 (공모전 제출 시점)
   공표 여부: 미공표 (PRIVATE 전환 완료)
   ```
3. 제출 파일: 소스코드 일부 (핵심 파일 선별 — 전체 제출 불필요)
   - `pipeline/scorer.py` (게이트 B 로직)
   - `ml/xgboost/model.py`
   - `backend/app/services/prediction_service.py`
4. 수수료: **2만원** (계좌이체 또는 카드)
5. 처리기간: 약 2~3주
