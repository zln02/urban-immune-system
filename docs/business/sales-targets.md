## 2. 1순위: 질병관리청 감염병감시지원단

### 2.1 도입 가설
KDCA ILINet (인플루엔자 표본감시) 는 임상 확진 기반이라 **1~2주 지연**. 우리는 약국 OTC + 하수도 + 검색으로 **선행 경보**.

### 2.2 데모 시나리오 (60초 Live)
1. **상황**: "강남구에서 KDCA 표본감시는 아직 GREEN, 그러나 우리 시스템은 YELLOW (Level 2)"
2. **트리거**: L1 약국 OTC 지수 78/100 (3주 연속 상승) + L3 검색 트렌드 65/100
3. **AI 리포트** (RAG): 2024-12-W1 강남구 유사 사례 인용, 권고 3가지 (백신·재고·방역)
4. **PDF 다운로드**: 4쪽 리포트 (감사 로그 포함)
5. **실측 근거**: Slide 9 의 F1 0.643 / Granger p<0.05 표

### 2.3 KPI 성공 기준
- 시범 90일 동안 **임상 확진 1주 이상 선행** 신호 발생 ≥ 3회
- 오경보 (FP) 월 1건 이하
- 감사 로그·ISMS-P 호환성 의견서 인계

### 2.4 가격
**Tier C 90일 = 3,000만** → 전환 시 Tier A Premium **연 1억~3억**.

## 3. 2순위: 서울시 감염병관리지원단

### 3.1 도입 가설
서울 25개 자치구 단위 모니터링 필요. 기존 DAESS·서울감염병포털은 **사후 집계** 중심.

### 3.2 데모 시나리오 (60초)
1. 25구 지도 (Folium · Okabe-Ito 팔레트)
2. 강남·송파·강서 3구 RED 경보 동시 트리거
3. SSE 스트림으로 17:23 갱신 → 모바일 푸시 시연 (PWA 시연 영상)
4. PDF 리포트 → 시민건강국장 메일 발송 자동화 (W19 작업 예정)

### 3.3 KPI
- 25구 전 지역 데이터 수집 안정성 ≥ 99% (60일)
- 시민건강국 의사결정 회의에 주 1회 자동 리포트 도달

### 3.4 가격
**Tier C 60일 = 2,000만** → 전환 시 Tier A Standard **연 6,000만**.

## 4. 3순위: WHO 협력센터

### 4.1 도입 가설
**한국형 사례연구**를 해외 협력센터(서울대 보건대학원·연세대 등) 에 데이터셋 + 코드로 제공 → 논문·정책 자문 채널 확보.

### 4.2 데모 시나리오
- KDCA ILINet vs 우리 3-Layer 비교 차트
- Granger 인과 p-value 표
- 학술용 라이선스 (CC-BY-NC) 데이터 패키지 1년치

### 4.3 가격
**Tier C 30일 = 1,000만** + 논문 공저권. 학술 우대.
