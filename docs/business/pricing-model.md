# 가격 모델 (3-Tier)

> v0.1 | 2026-04-14 | 박진영
> 초안. 실제 단가는 고객 반응·조달 기준·경쟁사 조사 후 확정.

## 개요
고객 유형별 3가지 tier. PoC 무상 → 전환 시 구독 또는 건당.

## Tier 1 — PoC (Proof of Concept)
- **대상**: 신규 고객 (초기 레퍼런스 확보 단계)
- **기간**: 3~6개월
- **요금**: 무상
- **범위**: 대시보드 읽기 전용 + 주 1회 리포트
- **목표**: 레퍼런스 확보, 고객 데이터로 모델 미세조정

## Tier 2 — 구독 (Subscription)
- **대상**: 상시 모니터링 필요 기관 (질병관리청·광역지자체)
- **요금**: 월 ₩XXX,XXX ~ ₩X,XXX,XXX (기관 규모별)
- **범위**:
  - 실시간 대시보드 (5인 내 동시 접속)
  - 경보 자동 발송 (이메일·SMS·Slack webhook)
  - API 호출 월 10,000건 포함
  - 월 1회 정기 회의 + 모델 업데이트
- **SLA**: 가동률 99.5%, RTO < 1h

## Tier 3 — 건당 (Per-Report)
- **대상**: 단발성 리포트 필요 (민간 보험·연구소·언론)
- **요금**: 리포트 1건당 ₩XXX,XXX
- **범위**: 특정 지역·기간·질병 대상 분석 리포트 PDF
- **납기**: 영업일 기준 5일

## 추가 옵션 (Add-on)
- API 추가 호출: ₩X/건 (월 10,000건 초과분)
- 신규 질병 모델 커스터마이징: 1회 ₩X,XXX,XXX
- 온프레미스 설치 (보안 기관): 초기 구축비 + 연간 유지관리 20%

## 경쟁 대비 포지셔닝
- vs. 의료 데이터 기반 솔루션: **의료 데이터 없이 작동 → 개인정보 부담 없음**
- vs. 해외 제품(BlueDot, Metabiota): **국내 3-Layer 데이터 소스 직접 연동, 한국어 UX**
