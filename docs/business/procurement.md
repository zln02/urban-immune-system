# 조달청·공공SW 등록 절차

> v0.1 | 2026-04-14 | 박진영

## 1. 사업자 자격

### 1.1 법인 설립
- [ ] 주식회사 또는 유한회사 설립 (캡스톤 발표 직후 가능)
- [ ] 사업자등록증 (업종: 소프트웨어 개발 및 공급업)

### 1.2 소프트웨어사업자 신고
- [ ] 한국소프트웨어산업협회(KOSA) 소프트웨어사업자 신고
- [ ] 중소기업 확인서 (중소기업현황정보시스템)

## 2. 조달청 등록

### 2.1 나라장터 (G2B) 가입
- [ ] 나라장터 기업회원 가입
- [ ] 공인인증서 등록
- [ ] 입찰참가자격 등록 (해당 업종)

### 2.2 조달청 종합쇼핑몰 등록 (선택)
- 공공기관이 바로 구매할 수 있는 카탈로그
- 다수공급자계약(MAS) 제도 활용

## 3. 공공SW 사업 참여 경로

### 3.1 직접 수주 (입찰)
- 나라장터에서 SW 사업 공고 모니터링
- 제안서 + 가격 제안 → 평가 → 낙찰
- 보통 2~3개월 프로세스

### 3.2 수요예보 대응
- 조달청 공공부문 SW 수요예보 (매년 초 발표)
- 해당 사업에 맞춰 제안 준비

### 3.3 혁신제품 지정 (우대)
- 조달청 혁신제품 지정 시 수의계약 가능
- 신청 요건: 기술성·혁신성·공공성 평가
- Urban Immune System 은 AI 혁신제품 후보로 유망

## 4. 인증 요구사항 (대형 사업 대비)

| 인증 | 필요성 | 예상 기간 |
|---|---|---|
| ISMS-P | 개인정보 처리 시 사실상 필수 (`isms-p-checklist.md`) | 6-12개월 |
| GS인증 | 공공SW 입찰 가점 | 3-6개월 |
| 국정원 보안적합성 검증 | 국가기관 납품 | 6-12개월 |
| CC인증 (EAL2+) | 국방·보안 기관 | 12개월+ |

## 5. 가점 전략
- 여성기업·청년창업기업 확인 (해당 시)
- 사회적기업 인증 (공공성 높음)
- 중소기업 확인서 (필수)
- 지식재산권(특허) 보유
