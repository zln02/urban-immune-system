- `backend/CLAUDE.md` + 루트 `CLAUDE.md` + 글로벌 `~/.claude/CLAUDE.md` 로드

## 7. 1등 진입 3대 핵심

### (1) 수치 정직성 복구 — 최우선, 데드라인 D+7
현재 README `F1=0.71, Precision=1.00, Granger p<0.05` 는 **계산되지 않은 하드코딩**. 심사자가 "계산 코드 보여주세요" 하면 즉시 붕괴.
→ sklearn `f1_score, matthews_corrcoef, precision_recall_curve` + statsmodels `grangercausalitytests` **실제 실행** 후 JSON 저장, README 교체. Precision=1.00 단독 표기 **제거** (MCC·AUPRC·N 병기).

### (2) 멀티 시즌 검증
단일 시즌 과적합 의심 방어. 2022-23, 2023-24 KDCA ILINet 공개 CSV 확보 → walk-forward CV 시즌별 결과표.

### (3) 라이브 데모 MVP
`docker compose up && streamlit run src/app.py` 로 15분 내 데모 가동 가능한 수준. 실데이터 미연결 시에도 graceful fallback.
