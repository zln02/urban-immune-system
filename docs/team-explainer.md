## 🎭 탐정 3명 비유 (핵심)

우리 시스템은 **탐정 3명이 동시에 의심**해야 경보 발령:

| 탐정 | 단서 | 비유 |
|---|---|---|
| 🕵 탐정 1 | 약국에서 감기약 많이 팔림 | "사람들이 증상 있어서 사간 것" |
| 🕵 탐정 2 | 하수구에서 바이러스 검출 | "무증상자까지 이미 감염된 것" |
| 🕵 탐정 3 | "고열 병원" 검색 급증 | "사람들이 불안해하는 것" |

**한 명만 외치면 = 드라마·뉴스 때문일 수도 있음 = 오경보**
**3명 동시에 외침 = 진짜 신호**

### 왜 이게 똑똑한가
Google 이 검색어 하나로 독감 예측하다가 2013년 실제보다 2배 과대추정해서 망함 (Google Flu Trends).
우리는 **하수(L2) 가 사람 행동과 완전 독립** 이라 거짓 경보 필터 역할. 이게 차별점.

## 🏭 공장 비유 (파이프라인)

우리 시스템은 **5단계 공장라인**:

```
[ ① 원료 수집 ]  →  [ ② 컨베이어 벨트 ]  →  [ ③ 냉장창고 ]  →  [ ④ 요리 ]  →  [ ⑤ 서빙 ]
```

| 단계 | 역할 | 실제 이름 | 우리 폴더 | 지금 상태 |
|---|---|---|---|---|
| ① 수집 | 신문 배달부 | Python 수집기 | `pipeline/collectors/` | 🟡 60% (코드만, API 키 없음) |
| ② 컨베이어 | 우체통 | Kafka | `pipeline/collectors/kafka_producer.py` | 🟡 40% (보내는 쪽만, 받는 쪽 없음) |
| ③ 냉장창고 | 시간 전용 창고 | TimescaleDB | `infra/db/init.sql` | 🟢 구조 완료, **row 0건** |
| ④ 요리 | AI 분석 | TFT + Autoencoder + RAG | `ml/` | 🟡 45% (코드만, 학습 안 함) |
| ⑤ 서빙 | 웨이터 | FastAPI + Streamlit | `backend/` + `src/` | 🟢 뜸 (시뮬 데이터로) |

### 지금 **단절된 지점**
```
수집 (60%) ──❌── 컨베이어 (40%) ──❌── 냉장창고 (0건) ──❌── 요리 (학습X) ──✅── 서빙
```
**② ③ ④ 가 연결 안 돼서** 지금 대시보드에 보이는 숫자는 **합성 데이터(가짜)**.

## 🔍 각 폴더 1문장 요약

**팀장이 팀원한테 이대로 읽어주면 됨**:

- `pipeline/` — "외부에서 데이터 긁어오는 로봇" (이우형 담당)
- `ml/` — "긁어온 데이터로 예측하는 AI 뇌" (박진영 담당)
- `backend/` — "뇌에서 나온 결과를 웹에 전달하는 배달원" (이경준 담당)
- `src/` — "공무원이 보는 그 화면 (Streamlit, Phase 1)" (김나영 담당)
- `frontend/` — "나중에 만들 고급 화면 (Next.js, Phase 2, 아직)" (김나영 담당)
- `infra/` — "이 모든 걸 돌리는 서버·CI 관리실" (박정빈 담당)
- `tests/` — "이상 없는지 검사" (박정빈 담당)
- `docs/` — "문서·회의록·포트폴리오" (박진영 + 전원)
- `analysis/` — "데이터로 F1 계산하는 노트북" (박진영)

## ❓ 팀원이 자주 할 질문 Top 10 + 팀장 답변

| Q | A |
|---|---|
| 이 파일 왜 있어요? | 그 폴더의 CLAUDE.md 읽어. "핵심 파일" 섹션에 한 줄씩 설명. |
| API 키 어디 있어요? | `.env` 파일. 없으면 `.env.example` 복사해서 만들고 박진영에게 값 요청. |
| 이 에러 어떻게 고쳐요? | 에러 전문 복사 → `cd <모듈> && claude` → Claude에게 붙여넣기. 30분 막히면 Discord. |
| 제가 다른 모듈 건드려도 돼요? | **안 됨.** 담당 디렉토리만. 필요하면 Discord에서 담당자 먼저 핑. |
| 테스트 어떻게 돌려요? | `pytest tests/ -q` (전체) · `pytest tests/test_foo.py -v` (하나만) |
| 제 코드 맞게 짜진 건지 | `cd <모듈> && claude` → "리뷰해줘, ruff 120자·타입힌트 규칙 맞는지" |
| git 꼬였어요 | `--force` 하지 마. `git status` → Discord 에 붙여넣고 박진영 호출 |
| Docker 안 떠요 | `sudo docker compose down -v && sudo docker compose up -d` → 안 되면 박정빈 |
| 브랜치 최신 아닌데 | `git fetch origin && git rebase origin/develop`. 충돌 나면 박진영 |
| 아예 막히면 | 해당 모듈 CLAUDE.md 다시 읽기 + Claude에게 물어보기 + Discord |
