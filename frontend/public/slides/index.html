전공 용어 8개 — 한 줄로 정리.

이 페이지의 한 줄을 기억하고 들으시면 뒤 슬라이드의 숫자가 다 이해됩니다.
① 3계층 (3-Layer)
약국 OTC · 하수 바이러스 · 검색 트렌드 — 사람이 병원 가기 전에 남기는 흔적 3종
② 시간 순 교차검증 (walk-forward)
AI 가 미래를 절대 못 보게 시간 순서로만 학습·평가. 점수가 낮아 보여도 진짜 점수.
③ 정확도 4종 (F1·Precision·Recall·FAR)
Precision = 경보 낸 거 중 진짜였던 비율.
Recall = 진짜 중에 잡아낸 비율.
F1 = 둘 합친 종합 점수.
FAR = 잘못 낸 경보 비율 (낮을수록 좋음).
④ 종합 위험도 (composite)
3계층 점수를 가중평균해 0–100 으로 만든 한 숫자. 30 · 55 · 75 넘으면 YELLOW · ORANGE · RED.
⑤ RAG (검색-증강 생성)
AI 가 답변 전에 WHO·KDCA 가이드 문서를 먼저 찾아본 뒤 그걸 인용해서 글을 씀. 환각 방지·출처 증명.
⑥ SSE (실시간 글자 전송)
서버가 브라우저로 한 글자씩 흘려보내는 방식. ChatGPT 화면에서 글자가 한 자씩 나오는 그 효과.
⑦ Granger 인과검정 / CCF
A 가 B 보다 먼저 움직이는지 통계로 검증. 우리 측정값: 검색이 임상 신고보다 3주 선행 (p=0.007).
⑧ B2G · PoC
B2G = 정부·공공 납품. PoC = 시범 운영 (소규모로 먼저 검증). 우리 목표는 질병관리청 B2G PoC.

# 코로나·인플루엔자·노로는 막대 색이 다름
def detect_bars(img, pathogen):
    low, high = COLOR_RANGES[pathogen]     # (R,G,B)
    mask = ((img >= low) & (img <= high)).all(axis=-1)
    return _segment_heights(mask)              # 막대별 높이 리스트

def parse_chart(img, pathogen):
    heights = detect_bars(np.asarray(img), pathogen)
    return heights[-1] / max(heights) * 100
    # 이번주 / 누적 max → 0~100 (KDCA 정의와 동일)
